# Overview Of Project

Objective

To build a ChatGPT-style AI chatbot using GPT-2.

To generate coherent and context-aware text responses.

To integrate AI with a web-based interface.

### Tools & Technologies

Programming Language: Python

Framework: Flask

AI Model: GPT-2

Library: HuggingFace Transformers, PyTorch

Frontend: HTML, CSS

### Working

User enters a message.

The message is processed by the GPT-2 model.

The model generates a contextual response.

Chat history is maintained with a sidebar.

### Key Features

Multiple chat sessions

Conversation history

Context-aware responses

Adjustable response quality
